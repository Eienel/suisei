Suisei MCP

Sui's agent toolkit. The Sui Stack as one-line tools, exposed over the Model Context Protocol so any AI agent can read, build, simulate, sign, and submit on Sui.

Remote endpoint

This deployment serves the MCP over Streamable HTTP at POST /api/mcp. It is a JSON-RPC API for MCP clients, not a web page. Add it to Claude as a Custom Connector (Settings, then Connectors). If SUISEI_MCP_TOKEN is set, callers must send Authorization: Bearer <token>.

Use it locally (Claude Desktop)

{
  "mcpServers": {
    "suisei": {
      "command": "npx",
      "args": ["-y", "@suisei-mcp/mcp"]
    }
  }
}

Packages

  • @suisei-mcp/mcp - the MCP server (33 tools)
  • @suisei-mcp/agent-signer - non-custodial local signer
